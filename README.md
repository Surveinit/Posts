### Installation
1. Clone this repository:
```bash
git clone https://github.com/Surveinit/Posts
```

2. Change directory
```bash
cd posts 
```

3. Install the required gems:
```bash
bundle install
```

4. Create DB:
```bash
rails db:create
```

5. Migrate DB:
```bash
rails db:migrate
```

6. Run the Rails server:
```bash
rails server
```
5. Visit the app in your browser:
```
http://localhost:3000
```
